# Browser State Lab

A no-build frontend exercise for local state, derived state, form validation, and browser storage.

## Run

From the repository root:

```bash
python3 -m http.server 5173 --directory projects/browser-state-lab
```

Open:

```text
http://localhost:5173
```

## What To Change

1. Add a new priority option.
2. Persist the selected filter in `localStorage`.
3. Add a second derived metric.
4. Replace the fake local data boundary with a real API call.
